# Segmentation Research Notes

This document tracks learning notes related to image segmentation models.

## Segmentation Types

- Semantic segmentation: classify every pixel into a category
- Instance segmentation: separate individual object instances
- Promptable segmentation: generate masks based on points, boxes, masks, or text prompts
- Open-vocabulary segmentation: segment concepts described by natural language

## Models of Interest

- EfficientSAM: lightweight segmentation experiments
- SAM-family models: promptable segmentation workflows
- SAM3: image/video segmentation and tracking research, subject to model access approval

## Responsible Use

Experiments should avoid collecting unnecessary user data. Local inference is preferred when possible, especially for camera and image applications.
