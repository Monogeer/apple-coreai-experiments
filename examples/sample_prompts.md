# Sample Prompts for Vision Experiments

These are example prompts for image segmentation or recognition experiments.

## Object Segmentation

- Segment the main object in the image.
- Find the person in the foreground.
- Segment the cup on the table.
- Detect and segment all visible cars.

## Object Tracking

- Track the selected object across video frames.
- Keep tracking the object even if it is partially occluded.
- Track all instances of the selected category.

## Image Understanding

- Describe the main objects in this image.
- Identify objects that could be interacted with in an AR scene.
- Find objects suitable for spatial anchoring.
